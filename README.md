# sysc4907
SYSC 4907 - Ava Wiebe, Jackson Fry, Stavros Karamalis - Road Condition Monitoring Platform

## Purpose
RoadMonitor is an IoT-Enabled Road Condition Monitoring Platform.
In-vehicle sensor modules collect and process data, then store it an online database accessible through the RoadMonitor API and website.

## Requirements:
FastAPI (Python)
SQLModel (Python)
libcurl (C++)
